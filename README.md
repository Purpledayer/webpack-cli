## Quick Start
#### Down
~~~
git clone https://github.com/bevelery/react-redux-starter-kit.git
~~~

#### Install
~~~
yarn install
~~~

#### Dll
第一次运行需要打包vendor，仅运行一次。当依赖(vendor)改变时需重新打包，更改index.html内引入的路径。
~~~
npm run dll
~~~

#### Run
~~~
npm start
~~~

#### Build
~~~
npm run build
~~~
